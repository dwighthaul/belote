# Belote @ Meltdown

## Objectifs
- Distribuer les tables selon le nombre de joueurs, en maximisant le nombre de personnes qui peuvent jouer
- Preformer les equipes sur chaque table de 4, 6 et 7
- Suivre en direct le nombre de personnes pretes a jouer
- Inciter les joueurs a se mettre en etat de "ready" pour jouer, afin de diminuer le temps entre chaque partie
